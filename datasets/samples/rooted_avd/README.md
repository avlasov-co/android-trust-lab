# Rooted AVD Sample

This directory contains synthetic / AVD-limited samples used to validate analyzer behavior across target-state and observer-privilege changes.

It includes both adb-shell and root-observer sample reports so target mutation and observer-boundary differences can be compared separately.

It does not claim physical-device boot-chain behavior, hardware-backed attestation behavior, TEE behavior, Widevine behavior, or OEM-specific bootloader behavior.
